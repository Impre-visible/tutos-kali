Faire du phishing

Dans ce tuto nous allons voir comment faire du phishing pour voler des identifiants

Avant de commencer, on aura besoin de quelques prérequis, telle que ce que nous propose ngrok on verra l'installation dans la prochaine partie.
Il nous faudra aussi un serveur web, peut etre fais sous apache2 ou autre, mais nous en parlerons aussi dans quelques instants, ainsi qu'un nom de domaine, vous pouvez en obtenir facilement avec quelques techniques dont je ne parlerez pas ici pour des soucis de légalités.
Il vous faudra aussi un template de site, facilement trouvable, pour ma part j'ai peu me procurer un pack avec une dizaine de sites gratuitement.

root@kali : cd ngork_apache/

root@kali : nano tuto_part1.txt

Pour commencer, créez un compte, et téléchargez le zip, puis extrayez le dans le dossier de votre choix.
Ensuite accédez à ce dossier dans votre terminale, en faisant cd votre_dossier/.
Puis on tape sudo mv ngrok /usr/bin/, puis pour vérifier que tous est installé, faites ngrok help. Vous devriez voir tout les options.
Maintenant on va installer Apache2, si vous avez un serveur web à distance prenez le, sinon faite le depuis votre machine, pour se faire faites, apt-get install apache2.
Et enfin, pour lancer le tous faites un petit service apache2 start et ngrok http 80. Voila, on a installé ngrok et apache.
Pour vérifier que tous va bien, dans votre moteur de recherche faite http://localhost ou tapez l'ip local de votre machine.
Enfin, ouvrez le port 80 de votre box.

root@kali : nano tuto_part2.txt

Maintenant installons la page de phishing, pour nous faciliter la vie on va le faire à la main, copiez la page de phishing de votre site désiré et placez la dans le dossier /var/www/html/ c'est la ou se trouve le dossier Apache2.
La page web principale doit s'appeler index.php.

root@kali : nano tuto_part3.txt

Voila vous avez votre site de phishing, vous pouvez l'envoyer par mail en utilisant un nom de domaine telle que faceb00k.fr ou autres.
Pour votre sécurité, je déconseille d'héberger le site de phishing chez vous, vous pouvez l'héberger chez des petits hébergeur qui ne font pas payer les premiers jours pour spammer vos victimes et ainsi ne rien payer.
