Faire du phishing (versio amélioré)

Dans ce tuto nous allons voir comment faire un phishing avancé en créant un faux réseau wifi. Ceci est a faire dans la rue.

Avant tous, il nous faut un logiciel, appelé wifiphisher, un acces à un ordinateur, et il faut que as carte wifi puissent se mettre en mode monitor, de plus il est recommandé d'etre dans un endroit ou les gens cherchent une connexion internet, telle que les aéroport, les fast-food ou encore les gares. Si vous vous trouvez dans un de ces lieux la, vous avez de très grande chance de trouver une victime.

root@kali : cd tuto_wifiphisher/

root@kali : nano tuto_part1.txt

On commence en installant wifiphisher, pour cela vous devais faire dans un terminale sudo git clone https://github.com/wifiphisher/wifiphisher
puis on accede au dossier d'installation en faisant un cd, je pense ne pas avoir à vous l'expliquer, puis on lance le setup.py sudo python setup.py install, voila, wifiphisher est installé, plus qu'a le lancer en faisant sudo wifiphisher.

root@kali : nano tuto_part2.txt

Maintenant, on sélectionne le wifi à dupliquer. On sélectionne le type d'attaque à utiliser, pour nous on sélectionne OAuth Login Page.

root@kali : nano Conclusion.txt

Voila c'est fini, plus qu'à attendre qu'une personne crédule n'accede à votre hotspot et s'y connecte, avec un peu de chance une personne se connectera en utilisant ces memes identifiants que son facebook ou son gmail, ce qui fais que vous avez la main mise.
